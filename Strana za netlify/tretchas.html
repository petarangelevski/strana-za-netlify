Tret Chas

Na tretiot chas uchevme za Listi i Tabeli.


Postojat ordered i unordered listi kako na primer:


  • Produkt 1
  • Produkt 2
  • Produkt 3

Ova e unordered lista i se koristi taka shto:

  • Otvarame ul tag
  • Vo ul tagot otvarame kolku sakame li tagovi, odnosno kolku ni se potrebni
Toa se pravi taka shto <ul>ovdeka se stavaat: <li>ovdeka vo li tagovite se nabrojuvaat predmetite </li></ul>

Ili pak ova:

  1. Produkt
  2. Produkt
  3. Produkt

Ova e ordered lista i se koristi taka shto:

  1. Otvarame ol tag
  2. Vo ol tagot isto taka otvarame kolku sakame li tagovi, odnosnno kolku ni se potrebni
Toa se pravi taka shto <ol>ovdeka se stavaat: <li>ovdeka vo li tagovite se nabrojuvaat predmetite </li></ol>

Tabelata se pravi taka shto:
se otvara tagot <table> pa vo tagot table se otvara table row tagot <tr> i tuka se vnesuva table heading zavisi kolku ni trebaat <th> Table Heading </th> pa se zatvara table row tagot </tr>
Pa se otvara nov table row tag <tr> pa tuka se otvara table data tagot zavisi kolku ni trebaat <td>Table Data</td> pa se zatvare table row tagot </tr>
pa tuka se zatvara table tagot </table>
