Shesti Chas - CSS

Na shestiot chas se zapoznavame so id, klasa, i kako se manipulira so niv vo CSS


Prvo za da gi povrzeme HTML so CSS vo head tagot se vnesuva:
<link rel="stylesheet" href="./styles.css">

ID i Class ne pravat razlika vo CSS bidejki na CSS ne mu e gajle kolku elementi go koristat ID-to, no vo JavaScript ima razlika zasto se koristat ID posebno i Class posebno
