ID Element

    ID se unikatni
  • Sekoj element moze da ima edno ID
  • Sekoja strana moze da ima eden element so toa ID

Id go definiriav na ul elementot taka shto:
<ul id="elementid1">
pa vmetnav li elementi
pa potoa go zatvoriv tagot so </ul>

Vo CSS file-ot se definira i manipulira so ID na elementot na primer:
#elementid1 { background-color: red; }
A crvenata boja e definirana samo na elementot chij ID go manipuliravme, vo slucajot toa e ul elementot, a li elementite ne se definirani
